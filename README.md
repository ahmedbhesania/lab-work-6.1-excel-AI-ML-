# 📊 Excel Project – List Comparison, Frequency Analysis & Text Functions

## 🚀 Project Overview

This project demonstrates how to work with **multiple lists, identify matching data, analyze repeating values, and generate abbreviations** using Excel formulas.
It focuses on practical techniques used in **data cleaning and analysis**.

---

## 🎯 Objectives

* Compare multiple lists and extract common or unique values
* Identify frequently occurring text values
* Create abbreviations using text functions
* Improve data manipulation skills in Excel

---

## 🧠 Topics Covered

---

## 📌 1. Compare Multiple Lists & Extract Matching Names

### 🔹 Description

Used to find **common and unique values between two or more lists**.

### 🔹 Techniques Used

* `FILTER()`
* `COUNTIF()`
* `UNIQUE()`
* `VSTACK()`

### 🔹 Applications

* Find employees in both lists
* Identify missing or unmatched data
* Create a consolidated unique list

---

## 📌 2. Find Most Occurring / Repeating Text

### 🔹 Description

Used to identify **frequent values in a dataset**.

### 🔹 Techniques Used

* `COUNTIF()`
* Pivot Tables

### 🔹 Applications

* Count employees per department
* Identify most common project
* Create frequency distribution

---

## 📌 3. Create Abbreviations using Text Functions

### 🔹 Description

Used to extract and combine text to create **short forms or abbreviations**.

### 🔹 Functions Used

* `LEFT()` → Extract first characters
* `TEXTSPLIT()` → Split text into words
* `TEXTJOIN()` → Combine text

### 🔹 Examples

* First letter abbreviation (IBM)
* First 2 letters of each word
* Custom formatted short names
